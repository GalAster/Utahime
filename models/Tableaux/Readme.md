# 词表法

---

所谓词表法, 说白了就是搞个数据库, 然后从里面抽就是了.

原始数据库 : https://github.com/chinese-poetry/chinese-poetry-zhCN

#### 函数: 


>UtahimeTableaux[long,str]
>> long 表示所有诗词的长度\
>> str 表示需要的藏头诗词


##### In: 
```mathematica
UtahimeTableaux[7,"苟利国家"]
```

##### Out:
```mathematica
苟非世尚且奇怪
利欲遮迷只自昏
国无峻柱可擎天
家中衣饭随精粗
```
